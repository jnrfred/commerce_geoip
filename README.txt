README file for Commerce GeoIP

INTRODUCTION
------------
This is a contribution to Drupal Commerce. This module adds the possibility to
show the country of an IP address of an order under the customer information section when viewing an order.

REQUIREMENTS
------------
This module requires the following:
* Submodules of Drupal Commerce package (https://drupal.org/project/commerce)
  - Commerce core,
  - Commerce Order (and its dependencies);
* GEOIP (https://www.drupal.org/project/geoip).


INSTALLATION
------------
* Install as you would normally install a contributed drupal module. See:
  https://www.drupal.org/docs/8/extending-drupal-8/installing-modules
  for further information.


HOW IT WORKS
------------

* It displays the Country of the IP address of an order when viewing an order under the customer information section on the rightside below the IP address
